Jeu d'Awélé:

Exécution du Jeu

Pour jouer au jeu, suivez ces étapes :

Lancement de l'Exécutable

Ouvrez l'exécutable game pour démarrer le jeu.

Si une erreur survient lors de la tentative d'exécution, veuillez suivre les étapes ci-dessous.

Construction de l'Exécutable
Ouvrez l'invite de commande (cmd).
Naviguez jusqu'au répertoire contenant le code source du jeu.
Tapez make et appuyez sur Entrée. Cela compilera le code source et créera un exécutable nommé game.

Comment Jouer
Démarrage du Jeu
Appuyez sur Entrée pour ouvrir le menu principal du jeu.
Choix du Mode de Jeu
Sélectionnez le mode de jeu que vous souhaitez jouer en naviguant dans les options du menu.
Jouer au Jeu
Pendant votre tour, sélectionnez un trou pour jouer votre coup.

Amusez-vous bien !
